### Project Overview
- Windows와 Mac에서 현재 사용자가 활성화 중인 Window의 정보 수집
- 현재 OS에서 CPU 사용율이 높은 Process Top 5 정보 수집


### Python Packages on Windows
```
- PyQt5
- pypiwin32
- psutil
```


### Python Packages on Mac
```
- PyQt5
- pyobjc
- psutil
```
